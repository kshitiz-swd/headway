#  Job Application Tracker

A full-stack job tracking application to manage applications, track progress, and gain insights into your job search.

## 🚀 Features

### 🧾 Application Management
- Add, edit, and delete job applications  
- Track company, role, platform, and status  
- Attach resumes to applications  

### 📄 Resume Management
- Upload and manage multiple resumes  
- Track resume performance  

### 📊 Dashboard
- Overview of applications, interviews, offers, and rejections  
- Monthly insights and trends  
- AI-generated insights and suggestions  

### 🔄 Pipeline (Drag & Drop)
- Move applications across stages  
- Real-time updates with backend sync  

### 📈 Analytics
- Applications over time  
- Platform-wise breakdown  
- Resume performance comparison  
- Status distribution  

### 🔐 Authentication
- Secure login/signup  
- Cookie-based authentication
